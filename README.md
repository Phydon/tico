# Tico

**time converter**

* calculate time delta between given timestamps

## Usage 

```shell
python3 tico.py <TIME> <TIME> <...>
```

## Example 

* calculate the time delta between 08:00 AM and 12:00 AM

    ```shell
    python3 tico.py 0800 1200
    ``` 
* output is 4 hours

    ```shell
    04:00
    ```
